# 2-Class Transition PRD

## Objective
Transition from 3-class (134 samples) to 2-class (115 samples) problem by extracting reversal samples to holdout set and retraining all models on the more balanced consolidation vs retracement classification task.

## Background
Current state: 3-class problem with imbalanced classes (consolidation: 65, retracement: 50, reversal: 19). The reversal class is small and ambiguous. Better to focus on the main binary classification problem.

## Tasks

### 1. Extract Reversal Holdout
Create extraction script to separate reversal samples from training data.
- Read current train.parquet (134 samples)
- Filter out reversal samples (19 samples)
- Save reversal samples to reversal_holdout.parquet
- Save remaining samples (115) back to train.parquet
- Verify class distribution after extraction

Implementation:
```python
import pandas as pd
from pathlib import Path

df = pd.read_parquet("data/processed/train.parquet")
print(f"Before: {len(df)} samples\n{df.label.value_counts()}\n")

reversal = df[df.label == 'reversal'].copy()
train = df[df.label != 'reversal'].copy()

train.to_parquet("data/processed/train.parquet")
reversal.to_parquet("data/processed/reversal_holdout.parquet")

print(f"After: {len(train)} train, {len(reversal)} holdout")
print(train.label.value_counts())
```
